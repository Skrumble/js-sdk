Department

Departments are groups of users that can all recieve calls to that department, depending on the ring strategy. For applications implementing call center applications (with many agents recieving incoming calls to one extension). Departments are used for calling groups only, for information about chats with multiple users, see Chat

new Department()

Creates a department

Departments are created with a list of users, which defines which users are eligible to recieve calls for this department.
